Plate This: Sourcing Tool

HOW TO USE

A step-by-step guide for food stylists — from building your sourcing list to exporting a formatted buy list ready for the market run.

Plate This · Food Styling Pre-Production Tool: photokitchenfood.github.io/sourcing
Overview

What this tool does

The Plate This Sourcing Calculator is a pantry cross-check tool. You type your sourcing list directly into the table — exactly how your team writes it — and the tool checks each item against the live Plate This pantry inventory. It tells you what's already in stock, what's running low, and what needs to be purchased. Then you export a formatted buy list.

🔗 Pantry inventory is always live The tool connects directly to the Plate This pantry Google Sheet every time it loads. Stock levels are always up to date — no manual syncing needed.
⚠ Use pantry units for accurate results Write quantities using the same units that appear in the pantry sheet — e.g. pcs, pack, bottle, kg, can, jar, bundle. Matching units helps the tool correctly identify what's in stock.
Step 01

Fill in project details

Open photokitchenfood.github.io/sourcing. Before doing anything else, fill in the three fields at the top of the page.

Project Name
The internal project name. e.g. Lady's Choice Campaign Q2
Client
The brand or client name. e.g. Unilever Philippines
Shoot Date
Click the field to open the calendar and pick the shoot date. This becomes the filename and tab name when you export.
⚠ These fields are required The tool will not run the pantry check unless all three are filled in.
Step 02

Type your sourcing list into the table

The table has five columns matching the sourcing categories your team already uses. Type one item per row in any column — exactly how you'd write it in your Google Sheet.

VEGGIES / FRUITS
Fresh produce, herbs, fruits. e.g. Beetroot, 2pcs · Cilantro, 2 packs · Orange, 3pcs
DAIRY / FROZEN
Eggs, milk, cream, cheese, frozen items. e.g. Eggs, 2 dozen
FRESH MEAT
Meat, poultry, processed meats. e.g. Chicken thigh fillet, 1kg
SEAFOOD
Fish, shrimp, shellfish. e.g. Shrimp, 500g
DRY GOODS
Pantry staples, condiments, baking items. e.g. Brown Sugar, 2kg · Cooking Oil, 2 bottles
1
Remove columns you don't need

Click × on any column header to hide it for this shoot. Use + Add column to bring it back.

2
Write manually or paste — both work

Type items one by one, or copy a list from your notes and paste into any column cell. Each line automatically drops into its own row.

3
Add more rows if needed

The table adds rows automatically as you fill the last one. You can also click + Add Row at the bottom.

Step 03

Check Against Pantry

Once your sourcing list is ready, hit the big orange button:

✦ Check Against Pantry →

The right panel switches to Step 1 — Pantry Check and shows every item with a status badge:

In Stock Low Stock Buy
Low Stock and not found = automatically added to Buy List Items flagged as Low Stock or not found in the pantry at all are automatically included in the Buy List when you generate it in Step 2. The team doesn't need to manually add these — the tool handles it.
You can re-run this anytime Made a change to the table? Just hit Check Against Pantry again — it always reads the latest values from the table.
Step 04

Generate the Buy List

Once you've reviewed the pantry check results, scroll to the bottom of the Step 1 panel and hit:

✦ Generate Buy List →

The tool switches to Step 2 — Buy List. This shows only the items that need to be purchased, grouped by category. If the tool detects similar items that could be combined (e.g. two types of oil), it will suggest a merge — the team can choose to combine or dismiss.

Step 05

Export your buy list

Go to Step 3 — Export using the tab at the top of the right panel. You have four export options:

1
↓ Download xlsx — recommended for Google Sheets

Downloads a formatted Excel file matching the standard Plate This sourcing sheet format — Consolas font, #FBBC04 yellow headers, six category columns. Import into Google Sheets via File → Import → Insert new sheet(s). The tab will be named with the client and shoot date automatically.

2
↓ Save as PDF / Print — for printed market lists

Opens a clean print preview in a new tab with the buy list formatted for A4. Hit 🖨 Save as PDF / Print to save or print. The file is named automatically as ProjectName_ShootDate_SourcingDayN. This is the recommended format for market runs.

3
↓ Export CSV — for spreadsheet import

Downloads a plain CSV with all buy list items, categories, and pantry stock levels. Good for importing into other tools or sharing via email.

4
Copy as Text — for messaging

Copies the full buy list as plain text to your clipboard. Paste directly into Viber, Slack, or any group chat for quick sharing with the team.

Step 06

Save your work and wrap up

After exporting, the What's next? section appears at the bottom of Step 3 with three options:

1
💾 Save & Start New Project

Saves the current sourcing list to the Saved Lists bar at the top of the page, then clears everything for a new project.

2
+ Add Another Sourcing Day

Saves the current day automatically, then opens a fresh empty table for Sourcing Day 2. Run the full check and export again for the next market day.

3
✓ Done for Now

Saves the project but keeps everything on screen as-is. Use this if you want to review before closing.

📋 Saved Lists — Shared Across All Stylists Saved lists are stored in the cloud and visible to the whole team regardless of which browser or device you're on. Each list shows who saved it. Anyone can load and edit — but only the creator can delete their own list.

Lists are organised by quarter and year based on the Shoot Date. Use the year buttons (2026 · 2027...) and quarter tabs (Q1 · Q2 · Q3 · Q4) to navigate. The tool always opens on the current year and quarter.

If you save a list with the same name, the tool asks: Replace (overwrite) or Save as New (creates V2, V3...).
