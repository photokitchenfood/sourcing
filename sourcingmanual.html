Plate This: Sourcing Tool

HOW TO USE

A step-by-step guide for food stylists — from extracting recipe ingredients using Claude or ChatGPT, to downloading your final buy list.

Tool: photokitchenfood.github.io Platform: Claude.ai or ChatGPT + Plate This Sourcing Tool
Overview

How the workflow works

The sourcing tool is split into two parts that work together. First, you use Claude or ChatGPT to read a recipe image or document and extract a clean ingredient list. Then you paste that into the Plate This Sourcing Tool which scales the quantities, checks against your live pantry inventory, and generates a formatted buy list you can export directly into your Google Sheet.

🔗 Pantry inventory is live The tool connects directly to your Plate This pantry Google Sheet every time it loads. Stock levels are always up to date — no manual syncing needed.
Step 01

Extract the recipe using an AI tool

Open any AI chat tool — Claude.ai or ChatGPT both work. Use the prompt below — save it somewhere accessible for easy reuse. Attach your recipe file (image, PDF, or pre-pro slide) and send.

✓ Works with both Claude and ChatGPT Use whichever you have access to. Both can read images, PDFs, and text. The extraction prompt works the same way on either platform.
EXTRACTION PROMPT — copy and use this every time: You are a food styling assistant for a production company called Plate This. Look at everything provided — this could be a recipe document, a pre-production slide, a photo of a dish, a PPM deck, or any combination. Your job is to extract a complete and practical ingredient list that the food styling team can use to source and prepare this dish for a photoshoot or videoshoot. Use all available clues: ingredient lists, dish descriptions, food styling notes, visible components in the photo, garnishes, sauces, serveware context, and any written notes on the slide. Output format: DISH NAME: [name of the dish] ORIGINAL SERVINGS: [number, or write "not specified"] INGREDIENTS: - [quantity] [unit] [ingredient name] (prep note if any) Rules: - List every ingredient including garnishes, toppings, sauces, and bread/base - If quantity is not stated, make a reasonable estimate for 1 serving, note as (estimated) - Use decimal numbers (0.5 not ½) - Keep ingredient names simple and clear - Flag brand name products with (BRAND ITEM) - If something is visible in the photo but not listed, include it and mark (from image) - Do not include cooking instructions — ingredients only
1
Attach your recipe source

You can attach a photo, screenshot, PDF, or pre-pro slide. If you have multiple pages for the same dish, attach all of them together — Claude will treat them as one recipe.

2
Paste the prompt and send

Paste the extraction prompt above into the chat along with your file. The AI will return a formatted ingredient list.

3
Review the output

Check that all ingredients are listed correctly. Any item flagged as (BRAND ITEM) is a client's product — you'll mark those as sponsored in the tool.

4
Copy the full output

Select and copy everything the AI returned — dish name, servings, and the full ingredient list.

Step 02

Open the Sourcing Tool and fill in project details

Go to photokitchenfood.github.io. Before doing anything else, fill in the three fields at the top of the page. These are required — they appear in your exported file.

Project Name
The internal project name. e.g. Lady's Choice Campaign Q2
Client
The brand or client name. e.g. Unilever Philippines
Shoot Date
Date and day of shoot. e.g. April 21, Tuesday — this becomes the tab name in your Google Sheet.
⚠ Don't skip these fields The Shoot Date field is used as the tab name when you import into Google Sheets. If it's blank, the tab will be unnamed and hard to find later.
Step 03

Paste the recipe and set scaling

1
Paste the extracted recipe text

In the left panel, paste the Claude output into the recipe text area. The tool reads the dish name and ingredient list automatically.

2
Set your scaling

Number of Takes — how many full batches the food styling team needs to prepare.
Recommended: 3 takes minimum for photoshoots, 5 takes minimum for videoshoots.

Meat Buffer (optional) — if meat portions need a different multiplier from takes.
Fresh Produce Buffer (optional) — if fresh items need a different multiplier.

3
Hit Parse & Scale Recipe

The recipe card appears below with all ingredients, their categories, original quantities, and scaled quantities. Review everything before moving on.

Step 04

Review the recipe card — mark sponsored items

The recipe card shows every ingredient with its auto-detected category and scaled quantity. Two things to do here before generating the master list:

1
Mark sponsored / brand items with ⭐

Any ingredient that is the client's product (e.g. Lady's Choice Real Mayonnaise for a Unilever shoot) should be starred. Click the ⭐ icon on that row. Starred items are tracked under the Client Provided tab and are automatically excluded from the buy list — the client supplies these directly.

2
Fix any wrong categories

Click the ✎ pencil icon on any row to edit the ingredient name, quantity, unit, multiplier, or category. The category dropdown contains the full list of pantry subcategories — make sure each item is in the right one so the buy list groups correctly.

⭐ About sponsored items Sponsored items appear only in the Client Provided tab — they never show up in the Master List or Buy List. This keeps your sourcing list clean and prevents the team from accidentally purchasing items the client is already providing.

If you have more than one recipe for the same project, paste each one in and hit Parse for each. They all stack into the same master list.

Step 05

Generate the Master List

Once all recipe cards look correct, go to the right panel and hit:

✦ Generate Master List →

The Master List shows every ingredient across all recipes, combined and de-duplicated, with its pantry status automatically checked:

In Stock Low Stock Buy ⚠ Check
🔗 Live pantry check The tool compares every ingredient against your live Plate This pantry inventory in real time. Green = you have it. Orange = needs to be purchased. Items not found in the pantry at all automatically go to Buy.

You can still edit items in the Master List — click ✎ on any row to adjust the name, quantity, unit, or override the status manually. This is useful when you know something is in stock but the system didn't match it correctly.

Step 06

Generate the Buy List

Once you're happy with the Master List, scroll to the bottom of that tab and hit:

✦ Generate Buy List →

The tool switches to the Buy List tab automatically. The list is grouped by:

  • Fresh Produce — first, since it's usually the first market stop
  • Frozen and Chilled Items — meat, seafood, dairy
  • Dry Goods: Cooking Items — sauces, condiments, spices, oils
  • Dry Goods: Pastry Items — flours, sugars, chocolates, nuts

Each section is broken down further by subcategory (e.g. Cooking Spices, Canned Goods, Meat and Poultry) so the market run is already organized.

Step 07

Export and upload to Google Sheets

1
Hit Download xlsx

Click the ↓ Download xlsx button. The file downloads with your project name, client, and shoot date in the filename. The button shows "⏳ Generating..." briefly while it builds the file.

2
Open the Plate This master sourcing sheet

Go to the quarterly Google Sheet where all buy lists are stored. The file is named by quarter — e.g. APR-JUNE 2026: PT Sourcing Lists. Make sure you're in the correct quarter's sheet for your shoot date.

3
Import the xlsx as a new tab

In Google Sheets: File → Import → Upload → select your downloaded file → choose "Insert new sheet(s)" → Import. The new tab will appear named with the client and shoot date you entered.

📋 What the file looks like The exported xlsx matches the standard Plate This format — PROJECT label in row 1, Client / Shoot Date / Budget in rows 2-4, then the six sourcing columns (VEGGIES/FRUITS · FRUITS · DAIRY/FROZEN · FRESH MEAT · SEAFOOD · DRY GOODS) with the yellow header row, Consolas font throughout.
💾 CSV backup If the xlsx download doesn't work, use the ↓ CSV button as a backup. You can import a CSV into Google Sheets the same way.
